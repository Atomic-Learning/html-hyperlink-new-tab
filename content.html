When creating a hyperlink using the <a> tag, you can provide attributes to cause clicking it to open in a new browsing context (usually a new browser tab). For example:

<a href="https://www.imperial.ac.uk" target="_blank" rel="noopener noreferrer">
    Visit Imperial College London
</a>

This link appears as "Visit Imperial College London", and clicking it will usually open the Imperial College London website in a new browser tab (or sometimes a new window, depending on browser settings).

Here we have used two attributes which serve different purposes. target="_blank" is the attribute which causes the link to open in a new browsing context, usually a new tab.

The rel="noopener noreferrer" attribute is a security measure which prevents the new page from being able to access the original page's window object, which can help to protect against certain types of cyber attacks. Specifying the rel attribute is not strictly necessary, but it is recommended for security reasons when creating hyperlinks that open in new tabs.
